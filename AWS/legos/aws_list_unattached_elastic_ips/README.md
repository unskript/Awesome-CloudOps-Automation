<h1>AWS List Unattached Elastic IPs </h1>

## Description
This Lego lists Elastic IP address and check if it is associated with an instance or network interface.


## Lego Details

    aws_list_unattached_elastic_ips(handle, region: str = "")

        handle: Object of type unSkript AWS Connector.
        region: Optional, AWS region. Eg: “us-west-2”

## Lego Input
This Lego take two inputs handle and region.
